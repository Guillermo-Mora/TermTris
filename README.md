<h1>The classic Tetris videogame on your terminal</h1>
<br />
<ul>
  <li>
    This project was made using the Java Lanterna libray, for creating and displaying terminal GUIs. <a href="https://github.com/mabe02/lanterna" target="_blank">Lanterna</a>
  </li>
</ul>
<br />

<h2>Requirements</h2>
<ul>
  <li>It can be executed on any terminal application and on any operating system that supports Java.</li>
  <li>Java version 21.0.9 or higher</li>
</ul>
<br />

<h2>How to play it</h2>
<ul>
  <li>Make sure you have Java 21.0.9 or higher.</li>
  <li>Download the "TermTetris.jar" file from this repository.</li>
  <li>Open your terminal and run the file: "java -jar TermTetris.jar"</li>
</ul>
